## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sholtodouglas/learning_from_play.git
   ```
2. Install pip packages
   ```sh
   pip install -r requirements.txt
   ```
